# AloneChat Workspace v0.1.0

> **这不是一个完美的版本，但这是一个诚实的开始。**

AloneChat Workspace v0.1.0 今天正式发布。

这是一个集成了实时聊天与 AI Agent 的协作平台。我们搭起了完整的骨架：ReAct 推理、多 Agent 编排、RAG 检索、MCP 市场、DeepSeek 百万级上下文优化——该有的都有了。

但我们不打算隐瞒它的不完美。代码里有未填的坑，有写着 TODO 的边界，甚至有已知的安全漏洞正等着修复。它们都躺在 `bugs/` 目录里，公开可见。

我们选择在这个时刻发布，是因为相信：一个项目的生命力不在于它发布时有多光鲜，而在于它发布后还能跑多快。接下来的每一天，我们都会修 Bug、补文档、打磨体验。

如果你愿意，欢迎一起见证它从 0.1 到 1.0 的过程。
