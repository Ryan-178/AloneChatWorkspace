 # AloneChat Workspace v0.1.2

> **从云端到本地，这一版让隐私真正落地。**

---

## 发布语

v0.1.1 填充了血肉，v0.1.2 扎下了根基。

这个版本我们做了一件大事：**让 AI Agent 回归本地**。

**本地部署优先**：新增 AloneChat CLI 工具，所有核心功能本地运行。用户代码永远留在本地，API 调用仅发送 Prompt，不发送代码。支持 Ollama 本地模型，完全离线可用。这是对隐私保护的极致追求——你的代码，只属于你。

**终端原生体验**：参考 Claude Code 设计，打造终端原生的交互体验。支持自然语言对话、代码生成、智能提交、自动测试。配合 Rich 库的精美 UI，让终端也能有 IDE 级的体验。


**配置管理革新**：API 密钥本地加密存储，配置文件 `.alonechatrc` 灵活管理，环境变量无缝支持。安全、便捷、可移植。

还有很多细节：上下文窗口控制、流式输出、多轮对话记忆……它们都躺在 commit 记录里。

依然不完美，但比 0.1.1 好了一点。这就是迭代的意义。
