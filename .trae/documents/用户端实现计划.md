# AloneChat 用户端实现计划

## 项目概述

为 AloneChat Workspace 创建一个现代化的用户端前端应用，用于与 AI Agent 框架进行交互。

## 技术栈

| 技术           | 版本     | 用途           |
| ------------ | ------ | ------------ |
| Next.js      | 16     | React 全栈框架   |
| React        | 19     | UI 库         |
| TypeScript   | 5.x    | 类型安全         |
| Tailwind CSS | 4      | 原子化 CSS      |
| shadcn/ui    | latest | 组件库          |
| Radix UI     | latest | 无头组件         |
| Lucide React | latest | 图标库          |
| Zustand      | latest | 状态管理         |

## 核心功能模块

**页面**:

* `/login` - 登录页面

**功能**:

* JWT Token 认证

* 登出功能

* 受保护路由

**API 端点**:

* `POST /api/auth/register` - 用户注册

* `POST /api/auth/login` - 用户登录

### 2. 聊天模块

**页面**:

* `/chat` - 对话列表

* `/chat/[id]` - 对话详情

**组件**:

* `MessageList` - 消息列表（虚拟滚动）

* `MessageItem` - 单条消息（支持 Markdown 渲染）

* `MessageInput` - 消息输入（支持文件上传）

* `ConversationList` - 对话列表侧边栏

**功能**:

* 实时消息收发 (WebSocket)

* 消息历史加载

* 对话创建/删除

* 文件共享

* 消息搜索

**WebSocket 消息类型**:

```typescript
interface WebSocketMessage {
  type: 'message' | 'typing' | 'read' | 'error'
  content: any
  timestamp: string
  metadata: Record<string, any>
}
```

**页面**:

* `/agent` - Agent 会话列表

* `/agent/[session]` - Agent 对话界面

**组件**:

* `AgentChat` - Agent 对话主组件

* `ThinkingBlock` - 展示 Agent 思考过程

* `ToolCall` - 展示工具调用及结果

* `ModeSwitch` - MTC/CODE 模式切换

**功能**:

* ReAct Agent 对话

* 实时思考过程展示

* 工具调用可视化

* MTC/CODE 模式切换

* 会话历史管理

**Gateway WebSocket 消息类型**:

```typescript
type GatewayMessageType = 
  | 'thinking'    // Agent 思考中
  | 'acting'      // 执行工具
  | 'observation' // 工具结果
  | 'final'       // 最终答案
  | 'error'       // 错误
  | 'task_progress'    // 任务进度
  | 'agent_thinking'   // Agent 思考过程
  | 'artifact_update'  // 产出物更新
  | 'mode_switch'      // 模式切换
  | 'skill_execution'  // Skill 执行状态
```

## API 集成详情

### Gateway WebSocket 连接

```typescript
// 连接 Gateway
const ws = new WebSocket('ws://localhost:18789/ws')

// 初始化
ws.send(JSON.stringify({
  user_id: 'user123',
  session_key: 'session-001'
}))

// 订阅频道
ws.send(JSON.stringify({
  action: 'subscribe',
  params: { channel: 'session:session-001' }
}))

// 发送消息
ws.send(JSON.stringify({
  type: 'message',
  body: '帮我计算 25 + 36 * 2'
}))

// 接收消息
ws.onmessage = (event) => {
  const data = JSON.parse(event.data)
  switch (data.type) {
    case 'thinking':
      // 显示思考过程
      break
    case 'acting':
      // 显示工具调用
      break
    case 'observation':
      // 显示工具结果
      break
    case 'final':
      // 显示最终答案
      break
  }
}
```
